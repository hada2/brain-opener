## Brain Opener

This is a tiny brainfuck interpreter showing cells created to solve UMDCTF 2022 [Trainfuck](https://github.com/sajjadium/ctf-archives/tree/main/UMDCTF/2022/rev/Trainfuck).


## Requirement

* Python 3
* curses
